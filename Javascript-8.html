Array Methods:-

(1).Length Method:-

Returns number of elements in array

(2)Push Method:-

Add elements at last position

(3)Pop Method:-

Remove element from last position

(4)Shift Method:-

Remove element from first position

(5)Unshift Method:-

Add element at first position

(6)Sorted Method:-

Sorts an array alphabetically

(7)Reverse Method:-

Reverses the elements in an array

(8)Splice Method:-

Adds new element into an array at any position

(9)Slice Method:-

Return a piece of an array

(10)Mapped Method:-

Creates a new array by performing a function on each array element

(11)Filter Method:-

Return array according condition

(12)Find Method:-

Return element according condition

(13)Every Method:-

If all elements follow the conditions then it returs true

(14)Some Method:-

If any one elements follow the conditions then it returs true

(15)Fill Method:-

Return an Array With All Elements by Filling same value
